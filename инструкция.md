## Необходимое ПО
На курсе понадобятся следующие утилиты/инструменты:
- python 3
- pip
- jupyter

Установка:

Python:
- Для Windows: https://www.python.org/
- Для Linux:
```shell
apt-get install python3
```

PIP
- Для Linux:
```shell
apt-get install python3-pip
```

Jupyter Notebook
Установка:
```shell
pip install notebook
```

Запуск:

- Linux:
```shell
jupyter notebook
```
